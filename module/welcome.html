This is the official documentation for the BTMC API Module.

What does our Module have?

Our module is the official BTMC api module, we implement full usage of the API and test new features with it. We have a promise and class based system to make it easy for new developers to understand how everything works aswell as implementation of some advanced features. Our Module also covers some things we may not allow other modules to monitor. Even better is that you can brag that you use the same module as the development team.
